The profile menu is the main account control surface in the authenticated app.

## Profile menu contents

- Profile settings
- Usage
- Billing and plan
- Theme
- Accounts
- Categories
- Bookmarks
- Import statement
- Shared access
- AI report
- Debt payoff
- Export CSV
- Print options
- Sign out

## Theme

New accounts default to dark mode. Users can switch between dark and light mode from the profile menu.

## Appearance settings

Users can configure:

- Transaction logos
- Brand colors
- Keyboard shortcuts
- Payday celebration
- Sidebar insights
- AI recommendations opt-in
- Marketing emails opt-in

## Usage

The usage modal shows plan limits and current usage. It is especially important for Free users.

## Export and print

Users can export the current month as CSV or print the calendar in a tidy or full format.
