Navigator Voice lets you add a transaction without opening the full transaction form first. You can speak a short command or type it into the Navigator Voice dock.

## Requirements

Navigator Voice requires:

- CalBudget Navigator access
- Navigator Voice enabled in settings
- An editable account
- Browser support for speech recognition if you want to speak commands
- Microphone permission if you use the mic

If your browser does not support speech recognition, you can still type a command into the dock.

## Start a voice command

You can start a command in three ways:

- Tap the mic button in the Navigator Voice dock
- Hold `V` while the calendar is focused
- Type a command into the dock and select Create

When using the keyboard shortcut, hold `V` to speak and release it to finish.

## Say or type a transaction

Use a short transaction command. For example:

```text
spent 24.18 on groceries today
```

Navigator Voice parses the command and creates a transaction in the active editable account. The created transaction includes notes that it came from Navigator Voice.

## What Voice is for

Navigator Voice is for transaction creation. It is not general dictation, and it is not a full chat replacement.

Use regular CalBudget Navigator chat when you want to ask questions about timing, low-balance days, or what to do next.

## Troubleshooting

If Voice does not work:

- Check that Navigator Voice is enabled in settings
- Check that your browser allows microphone access
- Try typing the command instead
- Make sure you are working in an editable account
- Confirm the calendar has an active account selected

If a transaction is created incorrectly, open the transaction and edit the date, amount, category, or notes.
