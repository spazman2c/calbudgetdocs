The debt planner helps users connect payoff goals to the calendar. A payment plan is only useful if the account can survive the timing.

## Debt account fields

- Name
- Debt type
- Balance
- APR
- Minimum payment
- Due day
- Linked account
- Status
- Notes

## Common debt types

- Credit card
- Personal loan
- Auto loan
- Student loan
- Medical debt
- Other debt

## Payoff planning

CalBudget can compare payment strategies and produce a schedule. Users can export planned debt payments back to the calendar so the payoff plan affects the running balance.

## Inferred debts

The app can infer possible debt accounts from recurring payments and transaction names. Inferred debts should be verified before relying on payoff calculations.

## Calendar-aware debt planning

Debt payments should be reviewed against:

- Paycheck dates
- Low-balance days
- Other fixed bills
- Upcoming subscriptions
- Savings transfers

## Paid access

Debt tools are included in Plus and Advisor.

<Note>
  CalBudget can model debt payoff scenarios, but it does not provide financial, legal, or credit advice.
</Note>
