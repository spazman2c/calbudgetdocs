The support inbox is where admins respond to user tickets.

## Ticket fields

- User ID when available
- Email
- Subject
- Message
- Status
- Admin notes
- Admin reply
- Admin replied timestamp
- Admin reply viewed timestamp
- User reply
- User replied timestamp
- Created and updated timestamps

## Ticket lifecycle

1. User opens a ticket.
2. User receives a confirmation email.
3. Admin reviews the ticket.
4. Admin replies from the support panel.
5. User receives a Resend email.
6. User sees an in-app ticket notification.
7. Viewed timestamp updates when the user opens the reply.

## Response expectations

The initial confirmation should tell users to expect a response within 24 hours.

## Reply rules

- Use the platform support flow.
- Reply-to should be `support@calbudget.com`.
- Keep notes separate from customer-visible replies.
- Do not include internal IDs unless useful to the customer.
- Close or update ticket status after resolution.
