## Admin sections

| --- | --- |

| Changelog | Release notes and announcements. |
| Documentation | Internal documentation entry point. |
| Audit log | Admin and sensitive actions. |

## Admin principles

- Prefer sync and reconcile tools before manual data edits.
- Keep user-facing email in Resend.
- Do not expose secrets in admin notes.
- Use audit logs for sensitive actions.
- Confirm identity before changing billing or access.

## Common support paths

- Subscription paid but not active: use subscription sync.
- iOS user missing: check RevenueCat aliases and sync all sources.
- User trial ended: confirm trial end date and Free limits.
- Email not delivered: inspect email events and Resend status.
- Ticket reply not seen: check ticket admin reply and viewed timestamp.
