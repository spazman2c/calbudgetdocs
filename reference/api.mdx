CalBudget is a Next.js app with route handlers under `/api`.

This is an internal route map, not a public API contract.

## Authentication

| Route group | Purpose |
| --- | --- |
| `/api/auth/*` | Web auth, login, logout, register, forgot password, reset password, social auth. |
| `/api/mobile/auth/*` | Mobile auth, Apple, Google/mobile callback support, session exchange, email/password login. |
| `/api/users/me` | Authenticated user profile and preferences. |
| `/api/users/usage` | Plan usage and limits. |
| `/api/users/presence` | User presence heartbeat. |

## Planning data

| Route group | Purpose |
| --- | --- |
| `/api/accounts` | Account CRUD. |
| `/api/categories` | Category CRUD. |
| `/api/category-rules` | Rule-based categorization. |
| `/api/transactions` | Transaction CRUD, search, groups, recurring scope behavior. |
| `/api/calendar-data` | Calendar payload for app shell. |
| `/api/forecast` | Forecast calculation. |
| `/api/bookmarks` | Saved transaction templates. |

## Planning tools

| Route group | Purpose |
| --- | --- |
| `/api/import` | Statement import and recurring detection. |
| `/api/reconcile` | Import row review, matching, and commit workflow. |
| `/api/debt/*` | Debt accounts, overview, plan, export. |
| `/api/insights` | Budget insights. |
| `/api/ai-recommendations` | Optional AI report and check status. |
