This page lists environment variable categories. Do not paste secret values into docs, tickets, or pull requests.

## Core

- `DATABASE_URL`
- `NEXTAUTH_SECRET` or app session secret equivalent
- `APP_URL` or deployment URL

## Auth

- Neon Auth base URL
- Neon Auth cookie secret
- OAuth client IDs and secrets where used
- Apple sign-in configuration

## Stripe

- Stripe secret key
- Stripe webhook secret
- Plus monthly price ID
- Plus annual price ID
- Advisor monthly price ID
- Advisor annual price ID

## Apple and RevenueCat

- App Store product IDs
- Apple shared secret or server notification configuration
- RevenueCat API key
- RevenueCat webhook secret
- Advisor product ID allowlists where needed

## Email

- Resend API key
- Resend webhook secret
- Email domain configuration
- Support email reply-to

## AI

- AI provider API key
- Model configuration
- Feature flags for AI recommendations or Advisor

## Plaid

- Plaid client ID
- Plaid secret
- Plaid environment
- Token encryption secret

## Widgets and analytics

- Live chat public key
- Analytics IDs
- Product Hunt or App Store links where shown publicly

<Warning>
  Never commit real secrets. Keep `.env.local` local and configure production secrets in the hosting provider.
</Warning>
