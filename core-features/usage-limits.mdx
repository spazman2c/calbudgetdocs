CalBudget has a free plan with clear limits and paid plans that remove those limits.

## Free plan limits

| Resource | Free limit |
| --- | --- |
| Accounts | 1 |
| Categories | 25 |
| Lifetime transactions | 50 |
| Recurring series | 5 |
| Bookmarks | 3 |

## Lifetime transaction count

The free transaction limit counts created transactions over the lifetime of the account. Deleting a transaction does not reduce the count.

This prevents users from cycling transactions to bypass the free plan.

## Paid plans

Plus and Advisor remove core usage limits:

- Unlimited accounts
- Unlimited categories
- Unlimited transactions
- Unlimited recurring series
- Unlimited bookmarks

Advisor also unlocks guided planning features.

## Usage modal

The profile menu includes a usage view that shows:

- Current plan
- Account usage
- Transaction usage
- Recurring series usage
- Bookmark usage
- Upgrade path when more room is needed

## Limit enforcement

When a free user exceeds a limit, the app should block the gated action and show an upgrade prompt. Reports, imports, debt tools, AI insights, and Advisor features are paid features.
