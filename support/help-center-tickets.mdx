CalBudget support lives in the help center, in-app ticketing, live chat, and email.

## Help center

Public help center:

[https://help.calbudget.com](https://help.calbudget.com)

## In-app help menu

The question-mark menu should link to:

- Help center
- Contact support
- Change history
- Keyboard shortcuts where relevant
- User tickets when the user has open or replied tickets

FAQ and video tutorial entries were removed in favor of Help Center.

## Ticket creation

When a user opens a support ticket:

1. The ticket is stored.
2. The user receives an email confirming it was opened.
3. The message tells them to expect a response within 24 hours.
4. The admin support panel shows the ticket.

## Admin replies

When an admin replies:

- The reply is saved to the ticket.
- The user receives an email through Resend.
- The user sees an in-app notification when they return.
- The ticket can be marked viewed after the user opens it.

## Support email

Use [support@calbudget.com](mailto:support@calbudget.com) everywhere.
