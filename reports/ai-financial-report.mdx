The AI Financial Report is a saved planning report generated from your CalBudget data. It is separate from CalBudget Navigator chat, checkups, and action plans.

<Note>
</Note>

## Before you generate a report

AI Financial Report generation is opt-in. CalBudget checks that your account has enough transaction data before it lets you generate a report.

You can choose:

- Past 3 Months to review recent spending history
- Next 3 Months to review scheduled upcoming transactions

For the past period, CalBudget expects at least five transactions per month and at least fifteen transactions total. For the future period, CalBudget expects enough upcoming transactions to produce a useful planning report.

## Generate a report

1. Open the AI Financial Report option.
2. Choose Past 3 Months or Next 3 Months.
3. Review the eligibility details.
4. Select Generate Report.

CalBudget analyzes the selected period and streams the report into the modal. When generation finishes, the report is saved so you can view it later.

## What the report covers

The report is designed to explain patterns and planning pressure in plain language. Depending on the selected period and your data, it may discuss:

- Income and expense patterns
- Category concentration
- Upcoming cash-flow pressure
- Low-balance dates
- Recurring obligations
- Practical planning considerations

The report uses the calendar data available at generation time. If you change transactions later, generate a future report after the refresh window opens.

## Refresh limit

CalBudget saves one AI Financial Report for three months at a time. The report footer shows when another refresh is available.

This limit keeps the report focused and prevents accidental repeated generation while you are still working from the same underlying data.

## Download the report

Use Download to save the report as Markdown. This is useful if you want to keep a copy with your own notes or compare it with future reports.

## Important limits

The AI Financial Report is informational planning help. It is not financial, legal, tax, or investment advice.

If the report seems wrong, check the calendar data first. Missing transactions, incorrect dates, or incorrect categories can change the report.
