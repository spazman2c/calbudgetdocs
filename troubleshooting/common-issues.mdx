Start with the simplest checks: refresh the page, confirm you are signed in to the right account, and make sure the date, account, and plan are correct.

## I paid but still see Free

1. Refresh CalBudget.
2. Sign out and sign back in.
3. Confirm you are using the same email or Apple sign-in identity used for the purchase.
4. Open **Billing and plan** from the profile menu.
5. Contact support from the live support widget if the plan still looks wrong.

Include whether you purchased on web or iOS.

## I cannot sign in

1. Confirm whether your account uses email/password, Google, or Apple.
2. If you use email/password, try **Forgot password**.
3. If you created the account with Apple or Google, use that same provider.
4. If the issue continues, contact support and include the sign-in method you tried.

## The calendar looks wrong

Check:

- The selected account.
- The selected month.
- The starting balance.
- Income vs expense type.
- Duplicate recurring transactions.
- Transactions assigned to the wrong date.
- Filters or search terms hiding transactions.

## I see text flash before the app loads

Refresh should show a small CalBudget loading state before the calendar appears. If you briefly see raw transaction text, contact support with your browser, device, and the page URL.

## A transaction amount has the wrong sign

CalBudget stores the amount as a positive number and uses **Income** or **Expense** to decide how it affects the balance.

- Use **Income** for paychecks or money coming in.
- Use **Expense** for bills, purchases, subscriptions, and payments.
- Typing `+100` can switch to income.
- Typing `-100` can switch to expense.

## Imported transactions look wrong

Imported files can use different sign conventions. Review each import before committing it to the calendar. If the file is a credit card statement, debits and credits may appear differently than a checking account export.

## The support widget is missing

Try:

1. Refresh the page.
2. Disable browser content blockers for CalBudget.
3. Try another browser.
4. Use [support@calbudget.com](mailto:support@calbudget.com) if the widget still does not appear.

## Advisor is unavailable

Advisor can sometimes show a fallback planning summary if the AI service is unavailable. The fallback is still informational and should not be treated as financial advice.
