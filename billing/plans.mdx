CalBudget has three access plans.

## Plan comparison

| Feature | Free | Plus | Advisor |
| --- | --- | --- | --- |
| Accounts | 1 | Unlimited | Unlimited |
| Lifetime transactions | 50 | Unlimited | Unlimited |
| Categories | 25 | Unlimited | Unlimited |
| Recurring series | 5 | Unlimited | Unlimited |
| Bookmarks | 3 | Unlimited | Unlimited |
| Calendar forecast | Yes | Yes | Yes |
| Imports and reconciliation | No | Yes | Yes |
| Reports and trends | No | Yes | Yes |
| Debt planner | No | Yes | Yes |
| AI recommendations | No | Yes | Yes |
| Advisor chat | No | No | Yes |
| Weekly checkups | No | No | Yes |
| Advisor actions and reminders | No | No | Yes |
| Shared access roles | No | No | Yes |

## Pricing

| Plan | Monthly | Annual |
| --- | --- | --- |
| Free | $0 | $0 |
| Plus | $2.99/month | $29.99/year |
| Advisor | $7.99/month | $79.99/year |

## Trial policy

The current product direction is free-first instead of trial-first. Users can start on Free, then upgrade when they need unlimited planning or Advisor.

## Entitlement sources

Access can come from:

- Manual admin access plan
- Stripe subscription
- Apple subscription
- RevenueCat subscription
- Temporary free access window

Manual admin access can override subscription status for support or testing.

## Gated features

Free users should be prompted to upgrade for:

- Extra accounts beyond the free limit
- Transactions after 50 lifetime creates
- More recurring series
- More bookmarks
- Imports and reconciliation
- Reports
- Debt planner
- AI recommendations
- Advisor features
- Shared access
