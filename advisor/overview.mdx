CalBudget Advisor is the guided planning tier. It includes everything in Plus and adds an AI-assisted planning layer around the user's forecast.

## Advisor features

| Feature | What it does |
| --- | --- |
| Advisor chat | Answers questions about the user's account forecast and upcoming cash-flow risk. |
| Weekly checkups | Generates a summary of risk, lowest balance, projected finish, and action items. |
| Action plans | Turns forecast signals into trackable next steps. |
| Reminder emails | Sends due-step reminders for open Advisor actions. |
| Shared access | Lets users invite another person with viewer, editor, or admin roles. |

## Advisor data context

Advisor uses:

- Account balance
- Recent transactions
- Future transactions
- Recurring items
- Top outflows
- Upcoming income and expenses
- Lowest balance date and amount
- Projected finish

## Access requirements

Users need Advisor entitlement from one of these sources:

- Manual admin override
- Stripe Advisor plan
- Apple or RevenueCat Advisor product
- Product ID configured as Advisor

Plus users do not automatically receive Advisor features.

## Product language

Use "Advisor" for the feature and plan. Avoid implying that CalBudget is a licensed financial advisor.

<Warning>
  Advisor provides informational planning help based on the forecast. It is not professional financial advice.
</Warning>
