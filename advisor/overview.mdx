CalBudget Advisor is the guided planning tier. It includes everything in Plus and adds an AI-assisted planning layer around your forecast.

<img src="/images/advisor-widget.svg" alt="CalBudget Advisor chat widget with quick questions and a forecast answer" />

## Advisor features

| Feature | What it does |
| --- | --- |
| Advisor chat | Answers questions about the user's account forecast and upcoming cash-flow risk. |
| Weekly checkups | Generates a summary of risk, lowest balance, projected finish, and action items. |
| Action plans | Turns forecast signals into trackable next steps. |
| Reminder emails | Sends due-step reminders for open Advisor actions. |
| Shared access | Lets you invite another trusted person with view, edit, or manage access. |

## Advisor data context

Advisor uses:

- Account balance
- Recent transactions
- Future transactions
- Recurring items
- Top outflows
- Upcoming income and expenses
- Lowest balance date and amount
- Projected finish

## Access requirements

Advisor features require the **Advisor** plan. **Plus** includes unlimited planning tools, imports, reports, debt tools, and AI recommendations, but it does not include Advisor chat, checkups, action plans, reminders, or shared access.

## Product language

CalBudget uses **Advisor** as the plan and feature name. It does not mean CalBudget is a licensed financial advisor.

<Warning>
  Advisor provides informational planning help based on the forecast. It is not professional financial advice.
</Warning>
