# CalBudget Docs

Mintlify documentation for CalBudget.

## Structure

- `docs.json` controls branding, navigation, navbar links, and global anchors.
- `*.mdx` files are documentation pages.
- `logo/` and `favicon.svg` contain docs branding assets.

## Local editing

Install and run the Mintlify CLI if you want a local preview:

```bash
npx mintlify dev
```
